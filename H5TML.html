Full cource HTML

Html is a language which is used to build a basic structure of any web page, it is able to view how
a web page looks. Its full form is "Hyper Text Markup Language.

  1. Tags

The main role in this language is of tags.
Without it you cant design any text you write. There are many of them, lets see some important tags that we have to use in every html code.

  1. "!DOCTYPE HTML" tag 
         It is a declaaration tag which declares that we are using html5(leatest version of html).
  1. "html" tag 
         It defines that the from where we are starting our html coding and where we gonna end
              this by opening and closing tags of html
  1. "head" tag 
         It is the head of the html code, it contains some meta information about the web page 
              for example a basic title tag which is used to display the title of the web page in 
              the browser's title bar.
  1. "body" tag 
         Everything which we want to display in the web page must be written under this tag 
              only, every heading; every paragraph; every image; etc.

  1. Attribute

Attribute in html is used to design or modify any text according to the developer of the web page.
there are many attributes such as colour, width, height, font, background color, etc.

Tables in HTML

Tables in HTML is not like others, it will not show the bordered part of the table or we can say blocks/cells.

      
Syntax of table tag-

          <.table.>
                                                               Ignore the full stops, it is used here to
            <.thead.>                                       show you the tag otherwise it will be used as a
                                                       tag and cannot be visible on the page!
                <.tr.>
                   <.th.>
                      title1
                   <./th.>
                   <.th.>
                      title2
                   <./th.>     
                <./tr.> 
            <./thead.>

            <.tbody.>
                <.tr.>
                   <.td.>
                      data1
                   <./td.>
                   <.td.>
                      data2
                   <./td.>
                <./tr.>
            <./tbody.>

          <./table.>

There are some tags related to tables:-

  • tr Defined as table row. This is used to display the row of the table. We can add as many data as we want to add in a single row tag.
  • td Defined as table data. This is used to display the data of the table.
  • th Defined as table header data. This is used to display the type of data we want to add in a table. Example Names, Class, Roll no. , etc.
  • thead This tag is used to indicate that we are going to add the heading names of the table by using 'th' tag inside this tag.
  • tbody This tag is used to indicate that we are going to add the data according to the header of the table using 'td' tag inside this tag.
  • table Table opening and closing tag to define from where we are giving the table data and to create table. All the tags related to table are used inside this tag only.

      • Note:
    1. to add a single header for 2 or more cells 'colspan="how many cells you want the same header"' Attribute under the 'th' tag.
    2. To add Title of the table you can use 'Caption'tag.

    Form in HTML

    forms are used to collect data from the users by sign up, login, requests, contact me, etc.



      Tags to be used while creating a form-

    • 'form' tag to start creating a file
    • 'input' tag to input any data. we can define type of data to be filled by 'type' attribute or type of inputting style,
      and 'placeholder' attribute to show some text on the blank where data has to be input.
    • 'label' tag is very useful for visually disabled persons so that they can listen what is written and what to do.
      Also it enable to select any option by clicking on the text. It has an attribute named 'id' which is used to let use the property of label who has the same id as in the attribute.


      Attributes to be used while creating a form-

    • 'action="any backend file path"' attribute used to define what action will be executed when a form is submitted.
    • 'class' attribute is used to make group of desired texts or paragraphs or titles anything.
    • 'id' attribute used to give an element a unique identification. It can't be only digits.
    • 'type' attribute used in 'input' tag, but there are many types of way to input data
      in simple word there are many type of options in 'type' attribute of the 'input' tag.
      • text it allows to enter any type of data in the blank inside 'input' tag.
      • password it also allow any type of data but it hides the text after writing it inside 'input' tag.
      • radio it is used when we have to select any one option like class or state etc inside 'input' tag. It has some attributes like value, name.
